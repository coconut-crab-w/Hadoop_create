# 系统设置

## 配置静态IP地址

### 1. 确认网络接口名称
>  1. `ip addr`: 可以全面了解当前系统的网络接口配置和状态。其中返回内容开头有网络接口名称，如：`eth0`、`wlan0`、`lo`等。
>  2. `lo` : 本机内部通信和测试 ,`<LOOPBACK,UP,LOWER_UP>` 这段信息中 UP表示接口已启用，LOWER_UP表示物理链路已连接（如网线已插好），这是网络连通的基础
>  3. `eth0` : 这个文件名不是固定的，根据你的网卡名称来，如：`wlan0`、`eth1`、`eth2`等。但是一般都是在`lo`下边。表示与外部网络进行通信，静态IP可以在这里设置 。
>  4.   文件路径：`/etc/sysconfig/network-scripts/ifcfg-eth0`

### 2. 编辑文件
>  1. 修改或添加以下内容（根据实际网络环境调整）：
> `BOOTPROTO=static
    ONBOOT=yes
    IPADDR=192.168.1.100
    NETMASK=255.255.255.0
    GATEWAY=192.168.1.1
    DNS1=8.8.8.8
    DNS2=8.8.4.4`
> 2. 说明：`BOOTPROTO=static` 表示使用静态 IP、`ONBOOT=yes` 表示开机启动

### 3. 重启网络
> 1. 保存文件后，重启网络：`service network restart` 或 重启服务器：`sudo reboot`

### 4. 验证配置
> `ip addr show ens33`、`ping -c 4 8.8.8.8`

## 关闭防火墙和SELinux
> 1. 关闭防火墙：`systemctl stop firewalld ` 或 `systemctl disable firewalld`
> 2. 查询防火墙状态：`systemctl status firewalld`
> 3. 禁用SELinux：
>    1. 什么是SELinux?在 Linux 系统中，SELinux（Security-Enhanced Linux）是一个内核安全模块，通过强制访问控制增强系统安全。但在某些开发、调试或兼容性场景下，可能需要将其永久关闭。
>    2. 编辑配置文件 `vi /etc/sysconfig/selinux`
>    3. 找到：`SELINUX=enforcing`
>    4. 修改为：`SELINUX=disabled`
> 4. 重启系统：reboot
> 5. 验证：`sestatus`
